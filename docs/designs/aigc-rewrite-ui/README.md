# 降低 AIGC UI 设计稿

该目录保存“降低 AIGC”功能的独立 UI 设计稿，暂不接入正式 `app/` 路由，便于后续评审和实现拆分。

## 文件说明

- `index.html`: 静态设计稿预览，包含桌面端主方案、移动端草图、关键状态和设计说明
- `styles.css`: 设计稿样式文件，使用独立视觉变量，不影响业务代码

## 设计重点

- 首屏直接解释“样本驱动改写”的工作方式，而不是展示模型参数
- 桌面端采用左侧录入、右侧结果的编辑台布局，保证结果区首屏可见
- 样本质量反馈前置到步骤 1，明确区分硬校验和软提示的视觉语义
- 结果区强调流式生成、thinking 折叠、复制和再次生成，符合高频迭代场景
- 移动端改成单列顺序，保留完整主流程但收敛信息密度

## 预览方式

直接在浏览器打开 [index.html](./index.html) 即可查看设计稿。
