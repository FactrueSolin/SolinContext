Design Notes

设计决策摘要

降低“AI 工具感”

采用纸张、批注、编辑台语义，弱化通用聊天产品气质，让用户更容易理解这是“仿写工作台”。

质量反馈前置

把样本可用性放进步骤 1 当场反馈，而不是等生成失败后再让用户回头找问题。

结果区更强存在感

桌面端右栏固定结果，保证流式输出和再次生成操作在同一视线区域完成。

移动端只保留主流程

在小屏下去掉复杂并排结构，按 1-2-3 顺序串起来，减少输入和阅读冲突。
